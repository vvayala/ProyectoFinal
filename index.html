Acerca de Nuestro Proyecto

Este proyecto tiene como objetivo analizar la satisfacción del cliente utilizando datos de retroalimentación. Actualmente, muchas empresas tienen dificultades para comprender las opiniones de sus clientes y mejorar sus servicios de manera eficiente. Esto demuestra que la falta de análisis adecuado para reducir la lealtad del cliente y afectar las ventas. Nuestra propuesta busca resolver este problema a través del uso de técnicas de Machine Learning y análisis de datos para identificar patrones en la satisfacción del cliente y sugerir mejoras basadas en estos hallazgos.

En esta pagina, presentamos algunos de los gráficos más relevantes que hemos generado a partir de nuestros análisis. Estos gráficos ayudan a visualizar tendencias, patrones y relaciones clave en los datos recopilados. Así tambien presentamos un modelo entrenado capaz de predecir la satisfacción del cliente en funcion de las caracteristicas proporcionadas.
